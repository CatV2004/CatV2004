### 🏙️ [Smart Urban Issue Management System](https://github.com/CatV2004/smart-city-backend)
**Full-stack & AI Developer** | *Feb 2026 – Jun 2026*

A smart city reporting system for handling large-scale citizen feedback with AI-powered issue classification.
- 🚀 Built RESTful APIs with **Spring Boot** + async processing with **Kafka** + caching with **Redis**
- 🗺️ Real-time map interface with **Next.js** for tracking report locations and statuses
- 💬 **WebSocket** for live status updates and event-driven notifications
- 🤖 AI-based image classification using **EfficientNet** and **CLIP** to auto-categorize urban issues
- 📍 Location-based assignment using geospatial data and distance calculation
- **Stack:** Spring Boot, Next.js, Kafka, Redis, WebSocket, PostgreSQL, EfficientNet, CLIP
- **Links:** [Backend](https://github.com/CatV2004/smart-city-backend) | [Frontend](https://github.com/CatV2004/smart-city-frontend) | [AI Service](https://github.com/CatV2004/smart-city-ai-service)

**Full-stack Developer** | *Oct 2025 – Dec 2025*

- ⚡ Performance optimization with **Redis** caching and **RabbitMQ** message queue
- **Stack:** Next.js, NestJS, FastAPI, PostgreSQL, Redis, RabbitMQ, Elasticsearch, NetworkX, ResNet50
